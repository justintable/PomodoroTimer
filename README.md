# 🍅 番茄钟 (Pomodoro Timer)

一个基于 Electron 的极简番茄钟桌面应用。

## 功能

- **25 分钟专注计时** — 圆形进度条可视化倒计时
- **5 分钟短休息 / 15 分钟长休息** — 每 4 个番茄钟触发长休息
- **开始 / 暂停 / 继续** — 空格键快捷操作
- **系统通知** — 计时结束时桌面提醒
- **番茄计数** — 记录已完成番茄钟数量

## 使用方法

```bash
# 安装依赖
npm install

# 启动应用
npm start
```

## 快捷键

| 按键 | 功能 |
|------|------|
| `Space` | 开始 / 暂停 / 继续 |
| `Esc` | 重置 |

## 技术栈

- Electron
- HTML / CSS / JavaScript
- SVG 圆形进度条
